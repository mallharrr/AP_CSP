# AP_CSP
AP CSP Create Task Project

## Purpose
This program helps users evaluate the strength of a password based on security rules like length, uppercase letters, numbers, and special characters.

## How It Works
The program asks the user to input a password, then checks it using a function called 'checkPassword()'. It assigns a score out of 5 and gives feedback based on the score.

## Key Features
- Input from the user via keyboard
- Output strength rating (Weak / Medium / Strong)
- Uses a list (ArrayList) for suggestions
- Uses a procedure ('checkPassword')
- Includes selection (if statements) and iteration (loops)

## How to Run
1) Compile:
   javac PasswordAnalyzer.java

2) Run:
   java PasswordAnalyzer
